# Social Media Resizer
A simple, fast, and modern web application that resizes any uploaded image to perfectly fit the recommended dimensions for all major social media platforms.

# Overview
Social Media Photo Resizer allows users to upload an image, choose a platform (Facebook, Instagram, Twitter/X, YouTube, Snapchat, etc), select the type of image (e.g., profile photo, cover photo, story, banner), and instantly download the image resized to the official best-fit dimensions.

The app uses high-quality image processing to ensure clean results while preserving aspect ratio.

# Instructions
1. Upload an Image
2. Select the social media photo dimension you want the image to be resized to
3. Preview the result and download
